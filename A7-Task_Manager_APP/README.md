# DOM Explorer Task Manager

## Project Overview

DOM Explorer Task Manager is a JavaScript DOM Manipulation project that allows users to manage tasks efficiently.

Users can:

* Add Tasks
* Edit Tasks
* Complete Tasks
* Delete Tasks
* Store Tasks in Local Storage
* Toggle Theme
* View Task Statistics

---

## Features

### Task Management

* Add new tasks
* Edit existing tasks
* Mark tasks as completed
* Delete tasks

### Local Storage

Tasks are saved in Local Storage so data remains available after page refresh.

### Statistics

* Total Tasks
* Pending Tasks
* Completed Tasks

### Theme Toggle

Users can switch between Dark Mode and Light Mode.

## Event Concepts

### Event Bubbling

Child → Parent → Grandparent

### Event Capturing

Grandparent → Parent → Child

### Event Delegation

A single event listener is attached to the parent container (`taskList`) to handle actions for dynamically created task cards.
