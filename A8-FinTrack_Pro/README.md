# FinTrack Pro - Personal Finance Tracker

A beautiful, responsive, and robust personal finance tracker built to help you track every rupee and build wealth daily.

## Features
- Track Income and Expenses
- View Monthly Summaries and Budget Alerts
- CSV Export and Print-friendly Reports
- Powered by Vite for fast development and optimized builds

## Setup Instructions

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Copy `.env.example` to `.env` and fill in your Supabase API keys:
   ```bash
   cp .env.example .env
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Build for production:
   ```bash
   npm run build
   ```
